在物理上如何对信息的编码进行表示

1 Voltage Representing

1.1 Encode a Picture

其中+1V表示白色，0V表示黑色，(0,1)间的电压值表示灰度值。其中图片的特定位置映射到序列上的某个点

对于这样的灰度图像最基本的处理结构块是复制和反转，其中反转只需进行 1 - v 就能得到其相反的值

上面的这个基于copy和invert的经过偶数次的反转，系统理论上得到的输出结果应该是与原图像完全一致，但是实际的输出变得模糊平滑

噪声导致了结果的不准确，经过的处理块越多，噪声对于结果的影响也会相应叠加，系统设计的目标则是在错误产生后并作为输入传入下一个处理块前尽可能的将其恢复。

1.2 Digital Abstraction

对于0/1编码问题，如果将 [0,1] 区间的电压值进行编码，将整个区间一分为2，界定一个边界 $V_{TH}$，其中

  • $\geq V_{TH}$ 的电压值认定为1
  • $<V_{TH}$ 的电压值认定为0

那么在一个十分微小的偏移量 $\mathcal{E}$ 的区间 $[V_{TH}-\mathcal{E},V_{TH}+\mathcal{E}]$ 中，使用真实设备检测这个边界条件耗时且价格昂贵，所以不可取

上面这种情况界定了两个边界，$V_L,V_H$，其中，

  • $\leq V_L$ 的电压值认定是 0
  • $\geq V_H$ 的电压值认定为 1
  • $(V_L,V_H)$ 之间的电压值是屏蔽区域

采用这种划分方式可以使用一个禁止区域内，使用一个放大器和屏蔽区的参考电压，参考电压不必十分精确，参考电压也可以根据温度不同而变化，只需要保证在划分的区域内能够正常分辨即可

1.3 Combinational Digital System

combinational device 需要满足以下规范

  • 信号输入，即转换器将电压值转换为数字化结果
  • 信号输出，将数字化结果转化为电压值
  • 能够满足将一个设备的输出作为输入传入另一个设备
  • 功能规范，即定义数字值和电压值的转换关系
  • 时序规范，即预计的处理时间

在上图中，输入总共有 $2\times 2\times 2$ 种，所以转换器需要将其变为 001 ～111 的编码形式，

如果要将几个小的combinational device 进行整合形成 system，需满足以下组成规则

  • 系统的每个 component 都是一个 combinational device
  • 每个 component 的 input 是系统的输入或者是其它 component 的 output
  • 系统中不存在环路

2 Noise

2.1 Generation

例如在两个 device 间一个的 output 作为另一个的 input ，output 的产生值是 $V_L-\mathcal{E}$，在传输的电缆中，因为噪声接收的 device 收到的信号值是 $V_L+\mathcal{E}$，是在 Forbidden 区域的，所以无法进行处理，因此在构建的系统中应该有相应的策略方法来避免噪声在信号传输中的影响

噪声的产生主要有以下几种原因

  • 电气特性决定，根据欧姆定律，IR 变化会导致电压变化
  • device 的组件参数和其标称值的微小差异引起
  • 来自外部电磁场的热噪声或者电压效应

device 的组成材料也会有相应的影响，这是不可避免的，好的解决方法是预测噪声可能产生偏差值，然后对其进行矫正

2.2 Noise Margin

一个好的解决方法是在输入端与输出端建立不同的规范，

  • digital output: “0” $\leq V_{OL}$, “1” $\geq V_{OH}$
  • digital input: “0” $\leq V_{IL}$, “1” $\geq V_{IH}$
  • $V_{OL}<V_{IL}<V_{IH}<V_{OH}$

通过在输出端两个边界加入一定的 margin，其中大小关系如上面的列表所示，左 margin 和右 margin 的大小不一定完全一致，其中较小的大小即这个 system 的抗噪能力

采用上述方法能够在噪声产生的影响逐渐积累前就将其消除，避免了因错误而产生的错误
